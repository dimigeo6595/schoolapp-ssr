A Spring Boot application designed for managing school operations. 
This project serves as a backend system with a web interface for handling school-related data.


Tech Stack
•
Java 21
•
Spring Boot 3.5.10
•
Spring Data JPA (for database persistence)
•
Thymeleaf (for server-side rendering)
•
Spring Web (MVC & REST API)
•
MySQL (Database)
•
Lombok (Boilerplate reduction)
Features
•
Web-based interface using Thymeleaf templates.
•
Database connectivity via Spring Data JPA and MySQL.
•
RESTful architecture support.
